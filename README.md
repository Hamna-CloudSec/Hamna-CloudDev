# 🐧 My Linux & Ubuntu Learning Journey

I have officially started learning Linux using the Ubuntu terminal. This repository is a record of the commands I am mastering and the progress I'm making.

## 🛠️ Commands I Learned in Class 1

| Command | Action | What it does (In plain English) |
| :--- | :--- | :--- |
| `pwd` | **P**rint **W**orking **D**irectory | Tells me exactly where I am in the folder system. |
| `ls` | **L**ist | Shows me all the files and folders inside my current location. |
| `cd` | **C**hange **D**irectory | Moves me into a different folder (like "opening" a folder). |
| `mkdir` | **M**ake **D**irectory | Creates a brand new folder with whatever name I choose. |
| `clear` | **Clear** Screen | Wipes the terminal screen clean so I can focus. |

---

### 📝 Notes from my first session:
* The terminal is case-sensitive (e.g., `Documents` is different from `documents`).
* Using `cd ..` (with two dots) takes me back one folder.
* I am using the Ubuntu terminal to navigate my computer without using a mouse.
